# JobHunterAI v0.8

AI-powered remote job hunter.

## Features

- Remote-only jobs
- Junior developer focus
- Resume scoring
- Telegram alerts
- AI recruiter
- AI cover letters
- Dashboard
- Docker
- GitHub Actions

## Setup

```bash
python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt
```

Copy

```
.env.example
```

to

```
.env
```

Run

```bash
python app.py
```
